## Objective
To implement image encryption and decryption using pixel manipulation techniques.

## Tools Used
- Python 3
- Pillow (PIL)

## Description
Image encryption is performed by modifying pixel RGB values using a key value.
The same key is used during decryption to restore the original image.

## How to Run
1. Place input.png in the folder
2. Run: python image_encrypt.py
3. Choose encrypt or decrypt

## Output
- encrypted.png
- decrypted.png
